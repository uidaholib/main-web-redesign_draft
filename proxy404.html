Due to a recent systems upgrade, the link you clicked is no longer valid. The URL of our proxy server has changed.

Most library proxy links can be repaired by replacing the old domain name http://ida.lib.uidaho.edu:2048/ with the new one https://uidaho.idm.oclc.org/. Information in the old link following the domain stays the same (note that new domain does not include a port number). For example:

  • Old link to Chronicle for Higher Education: http://ida.lib.uidaho.edu:2048/login?url=https://chronicle.com
  • New link: https://uidaho.idm.oclc.org/login?url=https://www.chronicle.com/

The library proxy server allows patrons to use U of I authentication to log in and access resources as if located on campus. Updated proxy links can be found in our Databases A-Z listings. Please contact our Reference Services (libref@uidaho.edu) for help.
