Journals

You can use this page to search and browse for journals and magazines by title and subject.

Abbreviated Journal Titles

You will need to know the full, unabbreviated journal title in order to search the UI Library catalog and determine if the UI Library owns a journal. If you don't know the full title, try one of the help pages linked below:

  • All That JAS: Journal Abbreviation Sources: a registry of Web resources that list or provide access to the full title of journal abbreviations.
  • Biological Journals and Abbreviations
  • ISI Journal Abbreviations Index
  • Journals Indexed in Agricola: agriculture database from the U.S. National Agriculture Library
  • MathSciNet Journal Search
  • PubMed Journals Database (Medline): can search abbreviations
  • WorldCat: a large database of library records (books, journals, etc.) that can be used for searching abbreviations.  Do a title search. Check the title you find carefully, since you may retrieve titles with some of the same words.
