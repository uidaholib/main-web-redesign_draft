The libarary makes available ArcGIS Server and ArcSDE for teaching, research, engagement, and administrative purposes. Please contact Bruce Godfrey for more information about utilizing these services.

 

Assistance

Have questions about implementing GIS for research or teaching? Need assistance with a GIS project? Contact Bruce Godfrey (bgodfrey@uidaho.edu/208.292.1407) to schedule an appointment.

 

ArcGIS Server

ArcGIS for Server is used to deliver and manage GIS, map, image, and data services across the enterprise and the web.

The library maintains ArcGIS Server sites that are available to students, faculty, and staff.

 

Enterprise Geodatabase (ArcSDE)

ArcSDE Technology is a core component of ArcGIS for Server. It manages spatial data in a relational database management system (RDBMS) and enables it to be accessed by ArcGIS clients.

The library maintains an ArcSDE geodatabase that is available to students, faculty, and staff.

 

ArcGIS Online

ArcGIS Online is a collaborative, cloud-based platform that allows members of an organization to use, create, and share maps, apps, and data, including authoritative basemaps published by Esri.

The University of Idaho ArcGIS Online Portal is available to students, faculty, and staff.

Please contact Bruce Godfrey for more information about utilizing these services.
